# research 스킬 설명

## 목적
- 조사·분석 결과를 단순 요약이 아니라 의사결정에 바로 쓸 수 있는 구조로 정리하기 위한 스킬이다.

## 핵심 규칙
- 질문 범위, 기간, 지역, 확실도 요구 수준을 먼저 정한다.
- 사실, 해석, 가설, 전망을 분리한다.
- 핵심 주장에는 근거 강도를 붙인다.
- 상반된 관점이 있으면 각 관점의 전제와 한계를 함께 적는다.
- 시간 민감한 주제는 날짜 기준을 분명히 적는다.

## 사용 시 주의
- 사실과 해석을 한 문단에 섞으면 안 된다.
- 최신성이 중요한 주제는 날짜 경계 없이 단정하면 안 된다.
- 바로 참고할 결론과 추가 검증 필요 항목을 분리해야 한다.
