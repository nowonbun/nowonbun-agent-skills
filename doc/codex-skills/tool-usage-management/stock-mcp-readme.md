# stock-mcp 스킬 설명

## 목적
- StockSearcher MCP를 쓸 때 목적에 맞는 툴을 고르고, 시장 코드와 날짜 파라미터를 먼저 검증하도록 만든 스킬이다.

## 핵심 규칙
- 종목 목록은 `list_stocks`, 시세 이력은 `stock_data`, 예측 기준일 목록은 `list_predict_dates`, 예측 결과는 `predict_rows`를 쓴다.
- `market`, `code`, `as_of` 같은 필수 파라미터를 호출 전에 확인한다.
- 시장이 여러 곳일 수 있으면 명시적으로 적는다.
- 결과 보고에는 어떤 툴을 왜 썼는지와 결과 의미를 같이 적는다.

## 사용 시 주의
- 예측 결과와 과거 시세 데이터를 섞어 설명하면 안 된다.
- 지원 시장 범위는 현재 MCP 응답으로 확인된 범위만 사용해야 한다.
