# 변경 내용

- 규칙 본문을 영어로 재작성하면서 기존 운영 의미를 유지했다.
- `# Flow`를 추가해 write-method selection → post-save verification → cross-review 순서를 명시했다.
- Source of Truth에 UTF-8 보존과 손상 진단 범위, `skill-modify-history` 위임 경계를 추가했다.
- 단일 주제 H2였던 `# Must NOT`와 `# Flow` 내부 제목을 제거하고 항목을 H1 직하로 이동했다.

# 재발 방지

- 구조 전환이 필요한 스킬은 Source of Truth만 덧붙이지 말고 skill-create-rule 전체 구조 기준으로 점검한다.
