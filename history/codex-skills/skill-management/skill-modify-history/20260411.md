# 변경 내용

- 기존 history 파일이 없을 때 기본 언어를 일본어에서 한국어로 변경했다.
- cross-review를 선택 조건이 아니라 mandatory phase 규칙으로 변경했다.
- `plan-review`, `source-review`, `result-review`, `re-review` 단계 요구를 추가했다.

# 재발 방지

- history 기본 언어 규칙을 수정할 때 상위 문서의 기본 문서 언어 정책과 함께 점검한다.
- cross-review 규칙을 수정할 때 applicability와 phase 목록을 함께 점검한다.
