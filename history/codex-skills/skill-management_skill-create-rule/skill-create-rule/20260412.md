# 수정 요약

- Source of Truth 규칙에 재사용 가능한 결정 경계 요구와 과적합 방지 조건을 추가했다.

# 원인

- 사용자는 Source of Truth에 단순한 다른 스킬 참조가 아니라 해당 스킬 고유의 정확한 근거가 드러나길 원했다.
- 기존 규칙은 문서, 결정, 위임 이유를 요구했지만 일회성 사례나 현재 개정 문맥에 과적합된 근거를 직접 금지하지 않았다.

# 변경 내용

- 외부 Source of Truth 참조가 해결해야 하는 대상을 `reusable decision boundary`로 명시했다.
- Source of Truth에 일회성 사례, 임시 예외, revision-local intent, workflow-local context를 근거로 쓰지 못하게 했다.
- Verification에도 같은 기준을 추가해 pass/fail 판단이 가능하게 했다.
- Korean companion readme를 새로 추가해 목적, 핵심 규칙, 사용 시 주의를 정리했다.

# 재발 방지

- Source of Truth를 수정할 때는 단순 참조 강화와 과적합 방지를 동시에 점검한다.
- 새 규칙을 넣을 때는 기존 항목과의 관계를 먼저 확인하고, 가능하면 기존 항목을 in-place 강화한다.
- Verification에는 Source of Truth 본문에서 새로 추가한 판별 기준을 그대로 반영한다.
