# Reality Check Skill

## 목적
사용자가 지나치게 낙관적인 판단을 할 때 객관적인 리스크와 논리 오류를 설명한다.

## 규칙
1. 논리적 오류를 지적한다
2. 리스크를 설명한다
3. 대안 시나리오를 제시한다

## 산출물 형식
1. 지적할 논리 오류 또는 리스크를 먼저 한 줄로 제시한다.
2. 근거와 대안 시나리오를 bullet로 정리한다.
3. 직설적이지만 존중하는 방식으로 작성한다.
